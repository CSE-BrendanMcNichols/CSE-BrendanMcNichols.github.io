Assignments

Assignment 1 – Basic HTML

I created a webpage about the Seattle Kraken with proper HTML structure, including headings, paragraphs, images, a list, a table, and links, all organized in the required directory structure.

Assignment 2 – Basic CSS

For Assignment 2 I created a new webpage about eagles with a navigation menu, a subheader, paragraphs, pictures, and a table, all styled with matching colors and formatted for a clean look.

Assignment 3 – Page Layout

I built a webpage using HTML5 elements and a flexbox layout with a responsive navigation bar. The design adjusts for different screen sizes with media queries and uses background colors, and padding to provide a clean look.

Assignment 4 – Recreate CSS Page

This assignment focuses on recreating a given webpage using HTML and CSS.

Assignment 5 – JavaScript Buttons, Functions, and More

Assignment 5 involves creating a dynamic webpage using JavaScript to add interactivity with buttons, random image refresh, and a slider to move elements around the page.

Assignment 6 – JavaScript: Conditionals

I implemented JavaScript conditionals to create dynamic, interactive web elements. The webpage includes a color slider, where the background color changes based on the slider value using an if statement to determine the RGB values. Additionally, I created a picture chooser feature that dynamically displays different images based on user input, using conditionals to select the appropriate image size.

Assignment 7 – JavaScript: Loops

This assignment focuses on using JavaScript loops to create an interactive star-drawing application, where users can input a number to generate stars, and receive feedback when clicking on each star.

Assignment 8 – JavaScript: Arrays

This assignment involves creating an interactive webpage where clicking on any of six provided images displays a title and description from an associative array, dynamically updated using JavaScript.

Assignment 9 – JavaScript: Classes

In this assignment, I created an interactive webpage using JavaScript classes to display information about planets. The page dynamically loads the data from an array of Planet objects, with a modal that reveals more information when clicking on each planet.

Assignment 10 – Parsing JSON

I created an interactive and responsive webpage that showcases a variety of ice cream flavors. The page connects to an external JSON file using async/await to dynamically load and display the ice cream flavors and images. I ensured that the layout is styled using modern CSS, including a hover effect that displays the flavor names over each image.
